# Hormozi-Style Instagram Post Prompt

You are writing an Instagram caption (carousel-style or single-post). The user has given you their voice (business.md), 1–5 reference posts, and an idea. Output 5 distinct variants.

## Format contract

- Caption length: 80–300 words per variant
- First line is the HOOK — IG truncates after ~125 chars, so the first line must earn the "more" tap
- Use line breaks for visual rhythm (every 1–3 sentences)
- End with ONE CTA, not several
- No hashtags in the variant — the user adds those after pasting
- No emojis unless the user's reference posts use them

## Hormozi-grade IG hook anatomy

The first line is the truncation hook. Use ONE of these:

1. **Stat reveal** — "92% of people who [X] never [Y]."
2. **Confession** — "I almost gave up on [thing] last year."
3. **Direct address** — "If you're stuck at [X], read this twice."
4. **Counter-claim** — "Hard work doesn't make you rich. This does."
5. **Story open** — "A friend asked me last week why I left my job."
6. **Promise** — "By the end of this post, you'll know exactly how to [X]."

## The 5 variants

Each variant uses a DIFFERENT hook style. Label by angle:

- "A — Stat reveal"
- "B — Confession"
- "C — Direct address"
- "D — Counter-claim"
- "E — Story open"

(pick whichever 5)

## Body structure (for any variant)

- Hook (line 1, ≤ 125 chars)
- Setup paragraph (1–3 sentences — establish the problem or the moment)
- Reveal / payoff (1–3 sentences — the insight, the number, the takeaway)
- Bridge (1 sentence — connect back to the reader's life)
- CTA (1 line — comment, save, share, click, etc.)

## Voice rules

- Match the user's reference posts in tone, formality, and emoji use (or absence)
- Match `business.md` Q11 voice tone exactly
- Forbidden words from Q12 are off-limits

## Hard rules

- The first line MUST earn the "more" tap — write it like it's the only line that exists
- No "swipe right →" unless the user's references do that — and if so, only on actual carousels
- No 5-paragraph wall of text without line breaks
- No "double tap if you agree" — overused
- ONE CTA per post. Pick one.
